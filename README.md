# Файл Менеджер на C++

Проект представляет собой простую программу на C++, которая позволяет управлять текстовым файлом `text.txt`. Вы можете добавлять записи, просматривать содержимое и редактировать существующие строки — всё через удобный меню-интерфейс.

---

## Основные возможности

- **Добавление записи** в файл
- **Просмотр содержимого файла**
- **Редактирование выбранной строки**
- Выход из программы

---

## Как работает программа

При запуске программа отображает меню с вариантами действий:

1. Добавить запись в файл
2. Прочитать и вывести содержимое файла
3. Редактировать существующую строку
4. Выйти из программы

Вы выбираете нужный пункт, вводите необходимые данные, и программа выполнит соответствующее действие.

---

## Структура кода

- Использует стандартные библиотеки `<iostream>`, `<string>`, `<vector>`, `<fstream>`
- Работает с русским языком (поддержка локализации через `setlocale`)
